# 드론 소음 예측기 v3

## 이번 수정 내용

- 예측 결과 카드의 `position: sticky`를 제거했습니다.
- 결과 카드가 스크롤 중 그래프 위에 겹치지 않고, 카드 아래에 그래프가 배치됩니다.
- 휴대폰 스피커에서 잘 들리도록 합성음의 주파수를 중음역 중심으로 조정했습니다.
- 소리 출력 크기를 높이되 최대 출력은 제한했습니다.
- 결과 카드에 체험 음량 70~140% 조절 기능을 추가했습니다.

## 주의

소음 듣기 기능은 예측된 실제 dB를 정확히 재현하는 기능이 아닙니다.
기기의 스피커 성능과 시스템 음량에 따라 실제로 들리는 크기가 다릅니다.
처음에는 기기 음량을 중간 정도로 설정하고, 이어폰보다는 스피커를 사용하세요.
