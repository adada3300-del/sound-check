# 드론 소음 예측기 v2

드론 종류와 비행 조건을 입력해 예상 소음 수준을 계산하는 모바일 친화적 단일 HTML 웹앱입니다.

## 주요 기능

- 예상 소음, 거리 감쇠량, 환경 보정값, 오차 범위 출력
- 고도에 따른 예상 소음 변화 그래프
- 예측값에 따른 상대적 드론 소음 4초 재생
- 오른쪽 상단 가이드 북
- 최근 예측 기록 자동 저장
- 기록 개별 삭제 및 전체 삭제

## 실행

`index.html`을 휴대폰이나 컴퓨터 브라우저에서 엽니다.

## GitHub Pages

저장소 최상단에 `index.html`을 업로드한 뒤 `Settings → Pages`에서
`Deploy from a branch`, `main`, `/root`를 선택합니다.

## 소음 듣기 주의

기기마다 스피커 성능과 시스템 음량이 다르므로 예측 dB를 실제 음압으로 정확히 재현할 수 없습니다.
소음 듣기는 예측값이 높을수록 상대적으로 크게 들리도록 만든 교육용 체험 기능이며,
웹앱 내부 출력은 낮게 제한했습니다. 재생 전에 기기 음량을 낮추고 이어폰보다 스피커를 권장합니다.
