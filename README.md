# 드론 소음 예측기

드론의 종류, 비행 고도, 수평거리, 프로펠러 수, 비행 상태, 주변 환경, 풍속을 입력하면
관측 지점에서 예상되는 소음 수준을 계산하는 모바일 친화적 웹앱입니다.

## 실행 방법

1. `index.html` 파일을 다운로드합니다.
2. 휴대폰이나 컴퓨터의 브라우저에서 파일을 엽니다.
3. 또는 GitHub 저장소에 `index.html`을 올리고 GitHub Pages를 활성화합니다.

## GitHub Pages 배포

1. GitHub에서 새 저장소를 만듭니다.
2. `Add file` → `Upload files`를 누릅니다.
3. `index.html`을 업로드하고 Commit합니다.
4. 저장소의 `Settings` → `Pages`로 이동합니다.
5. `Deploy from a branch`, `main`, `/root`를 선택하고 저장합니다.

## 계산 방식

- 기준 소음: 드론별 1m 거리의 교육용 대표값
- 거리 감쇠: `20 × log10(거리)`를 이용한 단순 자유음장 모델
- 프로펠러 수, 비행 상태, 주변 환경, 풍속을 보정값으로 반영
- 예상 오차 범위도 함께 표시

## 주의
